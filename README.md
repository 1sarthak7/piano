# Interactive Piano

A web-based interactive piano application that allows users to play songs and learn through a visual interface.

## Features

Virtual piano keyboard with playable keys

Visual guide for songs with falling notes

Audio playback utilizing the Web Audio API

Predefined songs to play along with

## Technologies Used

React

Vite

Web Audio API

JavaScript, HTML, CSS

## How to Run

Install dependencies using npm install

Start the development server using npm run dev

Open the provided local server URL in your browser
